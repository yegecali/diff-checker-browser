# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-09

### Added
- Initial release of HTML Diff Checker browser extension
- Visual element selection with click interface
- CSS selector manual input support
- Side-by-side diff visualization
- Regex-based pattern highlighting with color coding
- Persistent selector storage per domain
- Unified diff format copy to clipboard
- Keyboard shortcut support (ESC to cancel)
- Auto-opening side panel on diff generation
- Support for Chrome Extension Manifest V3

### Developer Experience
- ESLint configuration for code quality
- Prettier configuration for code formatting
- GitHub Actions CI/CD pipeline
- Comprehensive README documentation
- Contributing guidelines
- MIT License
- Pre-commit hooks with lint-staged
